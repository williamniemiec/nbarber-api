## ‚ùá Introduction
nBarber is an API built with Laravel Framework for a simple barber system. This application was made for the sole purpose of learning the Laravel framework better. You can interact with the project through the Railway platform ([click here to access](https://wniemiec-api-nbarber.up.railway.app/swagger-ui.html)).
